# KinoXP
## To contribute to the project do the following steps:
1. Clone the repo:
    ```
       git clone git@github.com:rimvydaszilinskas/kinoxp.git
    ```
2. Go to the folder:
    ```
       cd kinoxp
    ```
3. Add remote origin:
    ```
       git remote add upstream git@github.com:rimvydaszilinskas/kinoxp.git
    ```
4. Actually post some updates :)

### Happy coding!
